[bat파일?]
    1. 우리가 자동으로 실행 시키고자 하는 파일을 실행 시키고 프로그램이
    다운 됐을 때 자동으로 살려주는 역할.
    2. python 파일이 다운 됐음을 감지 못하는 경우가 있기 때문에 bat파일에 설정 된 횟 수 (ex. set max=1500)
    만큼만 카운트하고 다시 재시작 시키는 역할
    3. 구동 시키면 현재 실행 중인 python 프로그램을 모두 정지 시킨다.

[bat 파일 종류]
    * collector.bat : collector_v3.py 파일을 실행
    * trader.bat : trader.py 파일을 실행
    * bot_stop.bat : 현재 구동 중인 프로그램을 모두 종료

[collector 유의사항]
    * 15시 30분 이후(장마감 이후 돌리셔야합니다)
        collector를 배치 파일로 돌리며 데이터를 모을 때
        평일 기준 15시32분~23시59분 까지만 돌리는 것을 권장합니다.
        (주말, 공휴일은 00:01 ~ 23:59 가능)
        이유는 장시간(9시~15시30분)에 돌리면 종가 데이터가
        실시간으로 변하기 때문에 부정확한 데이터를 콜렉터가 가져 수 있습니다.
        따라서 작업스케줄러로 15시32분 쯤 시작하게 하고,
        23시 59분에는 만약 콜렉터가 구동 중이라면 자동으로 꺼지게 설정 하는 것이 좋습니다.

    * daily_craw DB는 따로 삭제 안해도 됩니다.

[작업스케줄러?]
    원하는 시간에 특정 프로그램 자동으로 실행 하기 위한 보조 프로그램

[작업스케줄러 등록 목적]
    * collector : 매일 평일 15:32에 collector_v3.py 파일을 실행 시킨다.
    * trader : 매일 장 시간 전 8:50에 trader.py 파일을 실행 시킨다.
    * bot_stop : 1. 장시간이 끝나면 15:31분에 trader 정지.
               2. 23:59분에 collector 정지
